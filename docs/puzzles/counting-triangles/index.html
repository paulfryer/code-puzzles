Counting Triangles

Problem Statement​

Given a list of N triangles represented by their three side lengths, determine how many distinct triangles there are. Two triangles are considered identical if they are congruent (same shape and size, regardless of orientation).

Function Signature: int CountDistinctTriangles(int[][] triangles)
Input: Array of triangles, each triangle represented as [side1, side2, side3]
Output: Number of distinct (unique) triangles

Example:

  • Input: [[2,2,3], [3,2,2], [2,5,6]]
  • Output: 2 (first two are the same triangle)
Real-World Applications

This pattern appears in:

  • Computer Graphics: Mesh deduplication in 3D models
  • GIS Systems: Identifying unique land parcels
  • Pattern Recognition: Shape matching in computer vision
  • Game Development: Collision detection optimization
  • Manufacturing: Quality control for triangular components

Key Insight​

The Sorting Strategy - A Visual Metaphor​

Think of triangles as puzzle pieces. Even if you rotate or flip a puzzle piece, it's still the same piece. Similarly:

Two triangles are congruent if their corresponding sides are equal, regardless of order.

  • [2,3,4], [3,4,2], and [4,2,3] represent the same triangle
  • Solution: Normalize triangles by sorting their sides, then use a HashSet

Implementation​

CountingTriangles.cs
using System;
using System.Collections.Generic;
using System.Linq;

class CountingTriangles 
{
    static void Main(string[] args) 
    {
        var test1 = new int[][] { new int[] {2, 2, 3}, new int[] {3, 2, 2}, new int[] {2, 5, 6} };
        Console.WriteLine(CountDistinctTriangles(test1)); // 2
    }

    private static int CountDistinctTriangles(int[][] triangles) 
    {
        var uniqueTriangles = new HashSet<string>();
        
        foreach (var triangle in triangles) 
        {
            // Normalize by sorting sides
            Array.Sort(triangle);
            
            // Create canonical string representation
            string normalized = $"{triangle[0]},{triangle[1]},{triangle[2]}";
            uniqueTriangles.Add(normalized);
        }
        
        return uniqueTriangles.Count;
    }
}

Optimized Implementation with Validation​

OptimizedCountingTriangles.cs
using System;
using System.Collections.Generic;
using System.Linq;

class OptimizedCountingTriangles 
{
    private static int CountDistinctTriangles(int[][] triangles) 
    {
        var uniqueTriangles = new HashSet<string>();
        
        foreach (var triangle in triangles) 
        {
            // Validate triangle inequality theorem
            if (IsValidTriangle(triangle)) 
            {
                // Sort to create canonical form
                Array.Sort(triangle);
                
                // Use custom hash for better performance
                string hash = $"{triangle[0]}_{triangle[1]}_{triangle[2]}";
                uniqueTriangles.Add(hash);
            }
        }
        
        return uniqueTriangles.Count;
    }
    
    private static bool IsValidTriangle(int[] sides) 
    {
        if (sides.Length != 3) return false;
        
        Array.Sort(sides);
        // Triangle inequality: sum of two smaller sides > largest side
        return sides[0] + sides[1] > sides[2];
    }
}
