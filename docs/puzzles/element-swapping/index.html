Element Swapping

Problem Statement​

Given a sequence of n integers and a limit of k consecutive swaps, determine the lexicographically smallest sequence achievable. Each swap can only involve adjacent elements in the current arrangement.

Function Signature: int[] FindMinArray(int[] arr, int k)
Input: Array of integers and maximum number of adjacent swaps allowed
Output: Lexicographically smallest array achievable

Example:

  • Input: [5, 3, 1], k = 2
  • Output: [1, 5, 3] (swap 3,1 then 5,1)
Real-World Applications

This optimization pattern appears in:

  • Sorting Algorithms: Bubble sort with limited iterations
  • Compiler Optimization: Instruction reordering with constraints
  • Task Scheduling: Priority-based scheduling with limited moves
  • Game Development: Puzzle games with move limits
  • Database Query Optimization: Join order optimization

Key Strategy​

The Greedy Bubble Approach - Visual Metaphor​

Think of it like organizing a line of people by height with limited moves:

Greedy Approach: For each position from left to right, find the smallest element within reach (k swaps) and bubble it to that position.

Step-by-Step Algorithm​

  • Visual Walkthrough
  • Execution Trace

Complete Example: [5, 3, 1, 2, 4] with k = 3​

Input: arr = [5, 3, 1], k = 2

Iteration 1: Position i = 0
  Remaining swaps: 2
  Search range: indices 0 to min(2, 2) = 2
  Elements in range: [5, 3, 1]
  Minimum: 1 at index 2
  
  Bubbling 1 from index 2 to 0:
    Swap arr[2] ↔ arr[1]: [5, 1, 3]
    Swap arr[1] ↔ arr[0]: [1, 5, 3]
  Swaps used: 2
  Remaining: 0

Iteration 2: Position i = 1
  Remaining swaps: 0
  Search range: indices 1 to min(2, 1) = 1
  Elements in range: [5]
  No swaps available, skip

Iteration 3: Position i = 2
  Already at last position, skip

Output: [1, 5, 3]

Implementation​

ElementSwapping.cs
using System;

class ElementSwapping 
{
    static void Main(string[] args) 
    {
        var result1 = FindMinArray(new int[] {5, 3, 1}, 2);
        Console.WriteLine($"[{string.Join(", ", result1)}]"); // [1, 5, 3]
    }

    private static int[] FindMinArray(int[] arr, int k) 
    {
        int[] result = new int[arr.Length];
        Array.Copy(arr, result, arr.Length);
        
        int remainingSwaps = k;
        
        for (int i = 0; i < result.Length && remainingSwaps > 0; i++) 
        {
            // Find smallest element within k swaps from position i
            int minIndex = i;
            int minValue = result[i];
            
            for (int j = i + 1; j < result.Length && j <= i + remainingSwaps; j++) 
            {
                if (result[j] < minValue) 
                {
                    minValue = result[j];
                    minIndex = j;
                }
            }
            
            // Bubble the minimum element to position i
            while (minIndex > i && remainingSwaps > 0) 
            {
                // Swap with previous element
                int temp = result[minIndex];
                result[minIndex] = result[minIndex - 1];
                result[minIndex - 1] = temp;
                
                minIndex--;
                remainingSwaps--;
            }
        }
        
        return result;
    }
}

Optimized Implementation with Better Tracking​

OptimizedElementSwapping.cs
using System;
using System.Collections.Generic;

class OptimizedElementSwapping 
{
    private static int[] FindMinArray(int[] arr, int k) 
    {
        int[] result = (int[])arr.Clone();
        int n = result.Length;
        int swapsRemaining = k;
        
        // Track swap history for debugging
        var swapHistory = new List<(int, int)>();
        
        for (int targetPos = 0; targetPos < n && swapsRemaining > 0; targetPos++) 
        {
            // Find the smallest element within reach
            int minIndex = targetPos;
            int minValue = result[targetPos];
            int maxReach = Math.Min(n - 1, targetPos + swapsRemaining);
            
            for (int j = targetPos + 1; j <= maxReach; j++) 
            {
                if (result[j] < minValue || 
                   (result[j] == minValue && j < minIndex)) 
                {
                    minValue = result[j];
                    minIndex = j;
                }
            }
            
            // Bubble the minimum element to targetPos
            while (minIndex > targetPos && swapsRemaining > 0) 
            {
                // Swap with previous element
                Swap(result, minIndex, minIndex - 1);
                swapHistory.Add((minIndex, minIndex - 1));
                
                minIndex--;
                swapsRemaining--;
            }
        }
        
        return result;
    }
    
    private static void Swap(int[] arr, int i, int j) 
    {
        int temp = arr[i];
        arr[i] = arr[j];
        arr[j] = temp;
    }
}
