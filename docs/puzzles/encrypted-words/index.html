Encrypted Words

Problem Statement​

Given a string s, generate an encrypted version using a specific algorithm:

  1. If length is 1, return the string as-is
  2. Otherwise, find the middle character (favor left for even lengths)
  3. Recursively encrypt the left substring, then the right substring
  4. Return: middle character + encrypted left + encrypted right

Function Signature: string FindEncryptedWord(string s)
Input: String to encrypt (1 ≤ length ≤ 10,000)
Output: Encrypted string using the recursive algorithm

Example:

  • Input: "facebook"
  • Output: "eafcobok"
Real-World Applications

This divide-and-conquer pattern appears in:

  • Cryptography: Message obfuscation techniques
  • Data Compression: Huffman encoding and tree-based compression
  • Search Algorithms: Binary search tree construction
  • Game Development: Procedural content generation
  • Bioinformatics: DNA sequence analysis

For "facebook" (length 8):

1. Middle index: (0 + 7) / 2 = 3 → 'e'
2. Left: "fac", Right: "book"  
3. Encrypt "fac" → middle='a', left="f", right="c" → "afc"
4. Encrypt "book" → middle='o', left="bo", right="k" 
   - Encrypt "bo" → "ob"  
   - Result: "o" + "ob" + "k" = "oobk"
5. Final: "e" + "afc" + "oobk" = "eafcoobk"

Wait, let me recalculate...

Correct trace:
1. "facebook" → mid index = 3, char = 'e'
   Left = "fac" (indices 0-2)
   Right = "book" (indices 4-7)

2. "fac" → mid index = 1, char = 'a'
   Left = "f" (index 0)
   Right = "c" (index 2)
   Result = "a" + "f" + "c" = "afc"

3. "book" → mid index = 1, char = 'o'
   Left = "b" (index 0)
   Right = "ok" (indices 2-3)
   
4. "ok" → mid index = 0, char = 'o'
   Left = "" (empty)
   Right = "k" (index 1)
   Result = "o" + "" + "k" = "ok"
   
5. "book" result = "o" + "b" + "ok" = "obok"

6. Final = "e" + "afc" + "obok" = "eafcobok"

The Divide and Conquer Pattern​

Implementation​

EncryptedWords.cs
using System;

class EncryptedWords 
{
    static void Main(string[] args) 
    {
        Console.WriteLine(FindEncryptedWord("abc"));      // "bac"
        Console.WriteLine(FindEncryptedWord("facebook")); // "eafcobok"  
    }

    private static string FindEncryptedWord(string s) 
    {
        // Base case
        if (s.Length <= 1) 
        {
            return s;
        }
        
        // Find middle index (favor left for even lengths)
        int middleIndex = (s.Length - 1) / 2;
        char middleChar = s[middleIndex];
        
        // Split into left and right substrings
        string left = s.Substring(0, middleIndex);
        string right = s.Substring(middleIndex + 1);
        
        // Recursively encrypt substrings
        string encryptedLeft = FindEncryptedWord(left);
        string encryptedRight = FindEncryptedWord(right);
        
        // Combine: middle + encrypted left + encrypted right
        return middleChar + encryptedLeft + encryptedRight;
    }
}

Optimized Implementation with Tracing​

EncryptedWordsWithTrace.cs
using System;
using System.Text;

class EncryptedWordsWithTrace 
{
    private static int callDepth = 0;
    
    static void Main(string[] args) 
    {
        Console.WriteLine("Encrypting 'facebook':");
        string result = FindEncryptedWordWithTrace("facebook");
        Console.WriteLine($"Final result: {result}");
    }
    
    private static string FindEncryptedWordWithTrace(string s) 
    {
        string indent = new string(' ', callDepth * 2);
        Console.WriteLine($"{indent}Encrypting: '{s}'");
        callDepth++;
        
        if (s.Length <= 1) 
        {
            callDepth--;
            Console.WriteLine($"{indent}  Base case: '{s}'");
            return s;
        }
        
        int middleIndex = (s.Length - 1) / 2;
        char middleChar = s[middleIndex];
        string left = s.Substring(0, middleIndex);
        string right = s.Substring(middleIndex + 1);
        
        Console.WriteLine($"{indent}  Middle: '{middleChar}' at index {middleIndex}");
        Console.WriteLine($"{indent}  Left: '{left}', Right: '{right}'");
        
        string encryptedLeft = FindEncryptedWordWithTrace(left);
        string encryptedRight = FindEncryptedWordWithTrace(right);
        
        string result = middleChar + encryptedLeft + encryptedRight;
        
        callDepth--;
        Console.WriteLine($"{indent}  Result: '{result}'");
        
        return result;
    }
}
