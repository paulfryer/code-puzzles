Nodes in a Subtree

Problem Statement​

Given a tree with N nodes where each node u corresponds to a character from string s (1-based indexing), answer Q queries. Each query [u, c] asks for the count of nodes in the subtree rooted at node u that contain character c.

Function Signature: int[] countOfNodes(Node root, ArrayList<Query> queries, String s)
Input: Tree root, Q queries of type [u, c], string s of length N with lowercase letters
Output: Integer array with response to each query

Real-World Applications

This tree query pattern appears in:

  • File System Analysis: Count file types in directory subtrees
  • Organization Hierarchies: Employee attributes within departmental trees
  • Biological Phylogeny: Species characteristics in evolutionary trees
  • Network Topology: Node properties within network segments
  • Compiler Design: Symbol table queries in nested scopes

Learning Objectives​

  • Master tree traversal algorithms (DFS) for subtree operations
  • Understand tree data structures and node representation
  • Learn efficient subtree query processing techniques
  • Practice mapping between different data representations
  • Apply preprocessing strategies for query optimization

Approach Breakdown​

Why DFS for Subtree Operations?​

Tree subtree queries naturally fit the Depth-First Search pattern:

  1. Subtree Definition: All nodes reachable from root through descendant edges
  2. Natural Recursion: DFS explores children before backtracking
  3. Complete Coverage: Ensures all subtree nodes are visited exactly once
  4. Memory Efficient: Uses call stack, no additional queue needed
  • ❌ Naive Approach
  • ✅ Optimized Approaches

Query-by-Query Tree Traversal: Traverse entire subtree for each query

// O(Q × N) - inefficient for multiple queries on same subtree
foreach (var query in queries) {
    int count = 0;
    TraverseSubtree(GetNode(query.nodeId), query.targetChar, ref count);
    results.Add(count);
}
// Problem: Repeated work for overlapping subtrees

Option 1: Preprocessing with Character Counts

// O(N + Q) - precompute all character counts per subtree
Dictionary<int, Dictionary<char, int>> subtreeCounts;
PrecomputeAllCharacterCounts(root);
// Answer queries in O(1) each

Option 2: DFS with Memoization

// Cache results for repeated subtree queries
Dictionary<(int nodeId, char c), int> memo;

Implementation​

Step-by-Step DFS Solution​

NodesInSubtree.cs
using System;
using System.Collections.Generic;

public class TreeNode 
{
    public int val;
    public List<TreeNode> children;
    
    public TreeNode(int val) 
    {
        this.val = val;
        this.children = new List<TreeNode>();
    }
}

public class Query 
{
    public int nodeId;
    public char targetChar;
    
    public Query(int nodeId, char targetChar) 
    {
        this.nodeId = nodeId;
        this.targetChar = targetChar;
    }
}

class NodesInSubtree 
{
    static void Main(string[] args) 
    {
        // Test case: Build tree from example
        //       1
        //      / \
        //     2   3
        //    /
        //   4
        
        var root = new TreeNode(1);
        var node2 = new TreeNode(2);
        var node3 = new TreeNode(3);
        var node4 = new TreeNode(4);
        
        root.children.Add(node2);
        root.children.Add(node3);
        node2.children.Add(node4);
        
        // Character string: s = "abac" (1-indexed)
        string s = "abac";
        
        var queries = new List<Query> 
        {
            new Query(1, 'a'),  // Expected: 2
            new Query(2, 'b'),  // Expected: 1  
            new Query(4, 'c')   // Expected: 1
        };
        
        int[] results = countOfNodes(root, queries, s);
        for (int i = 0; i < results.Length; i++) 
        {
            Console.WriteLine($"Query [{queries[i].nodeId}, '{queries[i].targetChar}'] = {results[i]}");
        }
    }
    
    private static int[] countOfNodes(TreeNode root, List<Query> queries, string s) 
    {
        // Build node mapping for quick lookup by ID
        var nodeMap = new Dictionary<int, TreeNode>();
        BuildNodeMap(root, nodeMap);
        
        // Process each query using DFS
        var results = new int[queries.Count];
        for (int i = 0; i < queries.Count; i++) 
        {
            var query = queries[i];
            var targetNode = nodeMap[query.nodeId];
            results[i] = CountCharacterInSubtree(targetNode, query.targetChar, s);
        }
        
        return results;
    }
    
    private static void BuildNodeMap(TreeNode node, Dictionary<int, TreeNode> nodeMap) 
    {
        if (node == null) return;
        
        nodeMap[node.val] = node;
        foreach (var child in node.children) 
        {
            BuildNodeMap(child, nodeMap);
        }
    }
    
    private static int CountCharacterInSubtree(TreeNode node, char targetChar, string s) 
    {
        if (node == null) return 0;
        
        int count = 0;
        
        // Check current node's character (convert 1-based to 0-based indexing)
        char nodeChar = s[node.val - 1];
        if (nodeChar == targetChar) 
        {
            count++;
        }
        
        // Recursively count in all child subtrees
        foreach (var child in node.children) 
        {
            count += CountCharacterInSubtree(child, targetChar, s);
        }
        
        return count;
    }
}

Optimized Preprocessing Solution​

OptimizedNodesInSubtree.cs
using System;
using System.Collections.Generic;

class OptimizedNodesInSubtree 
{
    private static Dictionary<int, TreeNode> nodeMap;
    private static Dictionary<int, Dictionary<char, int>> subtreeCharCounts;
    
    private static int[] countOfNodes(TreeNode root, List<Query> queries, string s) 
    {
        nodeMap = new Dictionary<int, TreeNode>();
        subtreeCharCounts = new Dictionary<int, Dictionary<char, int>>();
        
        // Build node mapping
        BuildNodeMap(root, nodeMap);
        
        // Precompute character counts for all subtrees
        PrecomputeCharacterCounts(root, s);
        
        // Answer queries using precomputed data  
        var results = new int[queries.Count];
        for (int i = 0; i < queries.Count; i++) 
        {
            var query = queries[i];
            if (subtreeCharCounts.ContainsKey(query.nodeId) && 
                subtreeCharCounts[query.nodeId].ContainsKey(query.targetChar)) 
            {
                results[i] = subtreeCharCounts[query.nodeId][query.targetChar];
            } 
            else 
            {
                results[i] = 0;
            }
        }
        
        return results;
    }
    
    private static void BuildNodeMap(TreeNode node, Dictionary<int, TreeNode> map) 
    {
        if (node == null) return;
        
        map[node.val] = node;
        foreach (var child in node.children) 
        {
            BuildNodeMap(child, map);
        }
    }
    
    // Precompute character counts for all subtrees using DFS
    private static Dictionary<char, int> PrecomputeCharacterCounts(TreeNode node, string s) 
    {
        if (node == null) return new Dictionary<char, int>();
        
        var charCounts = new Dictionary<char, int>();
        
        // Add current node's character
        char nodeChar = s[node.val - 1]; // Convert 1-based to 0-based
        charCounts[nodeChar] = charCounts.GetValueOrDefault(nodeChar, 0) + 1;
        
        // Add counts from all child subtrees
        foreach (var child in node.children) 
        {
            var childCounts = PrecomputeCharacterCounts(child, s);
            foreach (var pair in childCounts) 
            {
                charCounts[pair.Key] = charCounts.GetValueOrDefault(pair.Key, 0) + pair.Value;
            }
        }
        
        // Cache the result for this subtree
        subtreeCharCounts[node.val] = charCounts;
        
        return charCounts;
    }
}

Tree Flattening Solution​

For advanced optimization, flatten tree using DFS in/out times:

FlattenedTreeSolution.cs
class FlattenedNodesInSubtree 
{
    private static int[] countOfNodes(TreeNode root, List<Query> queries, string s) 
    {
        // Flatten tree using DFS traversal to get in/out times
        var inTime = new Dictionary<int, int>();
        var outTime = new Dictionary<int, int>();
        var dfsOrder = new List<int>();
        
        int timer = 0;
        FlattenTree(root, inTime, outTime, dfsOrder, ref timer);
        
        // Process queries using flattened representation
        var results = new int[queries.Count];
        for (int i = 0; i < queries.Count; i++) 
        {
            var query = queries[i];
            results[i] = CountInRange(dfsOrder, inTime[query.nodeId], 
                                     outTime[query.nodeId], query.targetChar, s);
        }
        
        return results;
    }
    
    private static void FlattenTree(TreeNode node, Dictionary<int, int> inTime, 
                                   Dictionary<int, int> outTime, List<int> dfsOrder, ref int timer) 
    {
        if (node == null) return;
        
        // Enter time for this node
        inTime[node.val] = timer++;
        dfsOrder.Add(node.val);
        
        // Visit all children
        foreach (var child in node.children) 
        {
            FlattenTree(child, inTime, outTime, dfsOrder, ref timer);
        }
        
        // Exit time for this node
        outTime[node.val] = timer - 1;
    }
    
    private static int CountInRange(List<int> dfsOrder, int start, int end, 
                                   char targetChar, string s) 
    {
        int count = 0;
        for (int i = start; i <= end; i++) 
        {
            int nodeId = dfsOrder[i];
            if (s[nodeId - 1] == targetChar) 
            {
                count++;
            }
        }
        return count;
    }
}

Memory-Optimized Solution​

MemoryOptimizedSolution.cs
class MemoryOptimizedNodesInSubtree 
{
    // Use iterative DFS to avoid recursion stack overhead
    private static int CountIterative(TreeNode root, int targetNodeId, char targetChar, string s) 
    {
        // Find target node first
        TreeNode targetNode = FindNode(root, targetNodeId);
        if (targetNode == null) return 0;
        
        // Iterative DFS using explicit stack
        var stack = new Stack<TreeNode>();
        stack.Push(targetNode);
        
        int count = 0;
        while (stack.Count > 0) 
        {
            TreeNode current = stack.Pop();
            
            // Check current node
            if (s[current.val - 1] == targetChar) 
            {
                count++;
            }
            
            // Add children to stack
            foreach (var child in current.children) 
            {
                stack.Push(child);
            }
        }
        
        return count;
    }
    
    private static TreeNode FindNode(TreeNode root, int targetId) 
    {
        if (root == null) return null;
        if (root.val == targetId) return root;
        
        foreach (var child in root.children) 
        {
            TreeNode found = FindNode(child, targetId);
            if (found != null) return found;
        }
        
        return null;
    }
}

Complexity Analysis​

Time Complexity​

ApproachTimeExplanation
Naive DFSO(Q × N)Each query traverses entire subtree
PreprocessingO(N + Q)Precompute once, answer queries in O(1)
Flattened TreeO(N + Q × subtree_size)Better for sparse queries
Iterative DFSO(Q × subtree_size)Same as recursive but avoids stack

Space Complexity​

ComponentSpaceNotes
Node MappingO(N)Store all nodes in hash map
PreprocessingO(N × 26)Character counts per node
Recursion StackO(height)DFS call stack depth
Query ResultsO(Q)Output array

Index Mapping Pitfalls​

// CRITICAL: Handle 1-based vs 0-based indexing correctly

// Node IDs are 1-based: node 1, node 2, node 3, ...
// String indexing is 0-based: s[0], s[1], s[2], ...

// Correct mapping:
char nodeChar = s[node.val - 1]; // Convert 1-based to 0-based

// Common mistakes:
char nodeChar = s[node.val];     // ❌ Index out of bounds!
char nodeChar = s[node.val + 1]; // ❌ Wrong character!
